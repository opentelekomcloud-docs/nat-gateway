SNAT Rule v2.0
==============

| Permission                       | API                                    | Action               |

| Creating an SNAT rule            | POST /v2.0/snat_rules                  | nat:snatRules:create |
+----------------------------------+----------------------------------------+----------------------+
| Querying SNAT rules              | GET /v2.0/snat_rules                   | nat:snatRules:list   |
+----------------------------------+----------------------------------------+----------------------+
